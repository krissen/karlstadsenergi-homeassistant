# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the project.

## Format

Each decision is documented in a file with format `NNNN-short-title.md`:

- 0001-api-authentication.md
- 0002-entity-design.md
- etc.

## Template

```markdown
# [Number]. [Title]

**Status:** [Proposed | Accepted | Rejected | Superseded by NNNN]
**Date:** [YYYY-MM-DD]

## Context

[What situation/problem does this decision address?]

## Decision

[What is the decision?]

## Source Material

[MANDATORY - Reference to research in research/_analys/]

- See `../research/_analys/[file]__primer.md`

## Consequences

[Positive and negative consequences of the decision]
```
